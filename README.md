# Gift Card Demo (Next.js)

This project is a Next.js (App Router) implementation of the Figma gift card redemption page.

## Scripts

- `npm run dev` starts the Next.js development server.
- `npm run build` creates a production build.
- `npm run start` runs the production server.

## Structure

- `app/page.tsx` contains the gift card page markup.
- `app/globals.css` contains the Figma-matched styles.
- `public/assets/figma/` contains exported SVG assets from Figma.
